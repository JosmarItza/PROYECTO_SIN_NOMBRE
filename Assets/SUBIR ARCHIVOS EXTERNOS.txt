SI AGREGASTE ARCHIVOS EXTERNOS DE UNITY (IMAGENES, AUDIOS, SCRIPTS, LOS CUALES NO CREARON UN .META POR TU PROYECTO DE UNITY) HACER LO SIGUIENTE

1. ANTES DE IMPORTAR LOS ARCHIVOS GUARDA: CTRL + S
2. IMPORTA TUS ARCHIVOS EXTERNOS A UNITY Y DESPUES EJECUTA ESTO (TODO ESTO SERA DE FORMA LOCAL):
	-> GIT ADD .
	-> GIT COMMIT -M "ARCHIVOS EXTERNOS GUARDADOS"
	-> GIT PULL

NOTA: ESTO SE DEBE DE HACER CADA VEZ QUE VAYAS A AGREGAR UN ARCHIVO EXTERNO DE UNITY
